# KingsCup

If Computer Scientists got invited to parties and needed a drinking game this is the kind of thing they'd use

[![Kings Cup Screenshot](./KingsScreenGrab.png)](https://owenkduffy.github.io/KingsCup/Kings.html)

Ditch the physical cards and embrace modernity with an online version of the classic party game! Gather a bunch of your friends and play a game of Kings using a simulated deck of cards and customisable rules so you can tailor the experience to your own tastes.

[Web version here](https://owenkduffy.github.io/KingsCup/Kings.html)
